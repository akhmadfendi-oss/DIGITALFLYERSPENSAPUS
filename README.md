# Undangan Digital Pelepasan Murid Kelas IX
## SMP Negeri 1 Pringapus (TA 2025/2026)

Aplikasi undangan digital premium dengan tema **Modern Javanese Elegance**.

### 🛠 Teknologi
- **React** (Vite)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

### 📂 Struktur Folder Assets
Untuk menjalankan aplikasi dengan sempurna, pastikan Anda menyiapkan file berikut di dalam folder `public/assets/`:

```text
public/
├── assets/
│   ├── logo/
│   │   └── logo-sekolah.png
│   ├── music/
│   │   └── backsound.mp3
│   └── gallery/
│       ├── foto-1.jpg
│       ├── foto-2.jpg
│       ├── foto-3.jpg
│       ├── foto-4.jpg
│       ├── foto-5.jpg
│       ├── foto-6.jpg
│       ├── foto-7.jpg
│       └── backdrop.jpg
```

*Catatan: Jika file gambar belum tersedia, aplikasi akan otomatis menampilkan placeholder elegan.*

### 🚀 Cara Menjalankan Project

1. **Instalasi Dependency**
   Buka terminal di folder project dan jalankan:
   ```bash
   npm install
   ```

2. **Jalankan Mode Development**
   ```bash
   npm run dev
   ```

3. **Build untuk Produksi**
   ```bash
   npm run build
   ```

### ✨ Fitur Utama
- **Loading Screen**: Animasi pembuka elegan dengan shimmer emas.
- **Background Batik**: Pattern batik halus menggunakan CSS.
- **Gunungan Wayang**: Ilustrasi Gunungan menggunakan inline SVG (bebas ketergantungan file).
- **Music Player**: Tombol floating untuk memutar backsound.
- **Glassmorphism**: UI modern dengan efek blur kaca dan border emas.
- **Responsive**: Mobile-first design, sangat rapi di HP maupun Laptop.
