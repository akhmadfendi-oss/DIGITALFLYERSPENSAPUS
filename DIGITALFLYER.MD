# Digital Invitation Specification: "Elegansi Nusantara"

## 1. Project Overview
Membuat undangan digital premium dengan tema "Modern Javanese Elegance" (Hitam & Emas) yang responsif, interaktif, dan memiliki kesan mewah.

## 2. Visual Identity
- **Primary Colors**: Deep Black (#0a0a0a), Royal Gold (#d4af37), Cream (#f5f5dc).
- **Typography**: 
  - Serif (Playfair Display) for headings.
  - Sans-serif (Inter/Outfit) for body text.
- **Graphic Elements**: 
  - Gunungan Wayang (Gold Gradient).
  - Ornamen Batik Parang/Kawung.
  - Glassmorphism effects for cards.
  - Smooth parallax & scroll animations.

## 3. Core Features
- **Hero Section**: Countdown timer, background video/animasi gunungan, judul acara.
- **Invitation Card**: Nama penerima (dinamis via URL), kata pengantar.
- **Event Details**: Waktu, lokasi (Google Maps integration), timeline acara.
- **Gallery**: Slider foto kenangan/pre-event.
- **RSVP & Guestbook**: Form konformasi kehadiran.
- **Admin Panel**: Client-side editing (LocalStorage) untuk teks dan gambar.

## 4. Tech Stack (Vanilla)
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flex/Grid, Keyframes animations.
- **JavaScript**: ES6+, ScrollReveal, GSAP (optional via CDN), Lucide Icons.

## 5. Implementation Workflow
1. **Foundation**: CSS variables, resets, and utility classes.
2. **Layout**: Responsive grid system.
3. **Components**: Buttons, Cards, Modal, Slider.
4. **Interactivity**: Animations and dynamic data handling.
5. **Polishing**: SEO, accessibility, and performance optimization.
